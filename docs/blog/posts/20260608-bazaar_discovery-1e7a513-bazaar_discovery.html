How x402 fits agent-to-agent commerce

HTTP 402 Payment Required turns every API route into a priced SKU. Agents receive a payment challenge, settle in USDC on Base, XRP on XRPL via T54, or CELO on Celo, then retry with proof—no accounts, no OAuth handoffs, no speculative trading.
