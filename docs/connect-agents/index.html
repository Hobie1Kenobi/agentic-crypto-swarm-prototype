Quick start

  1. Read MCP integration (Cursor / Claude Desktop / Smithery).
  2. Point MCP at https://api.agentic-swarm-marketplace.com/mcp (Streamable HTTP).
  3. Configure x402 buyer credentials per repo .env.example (CDP keys for Base USDC).
  4. Expect HTTP 402 on unpaid GET — pay, then retry for JSON results.
  5. Load live routes from endpoints.json and /.well-known/x402.json on the API host.

Payment rails

  • Base USDC — /x402/v1/* (commerce data, research brief, contract triage, ecosystem-pulse)
  • XRPL (T54) — /t54/x402/v1/* (XRP drops, same SKU shapes)
  • Celo CELO — /celo/query native fulfillQuery

For humans vs autonomous agents

Humans browse this site and the pulse board. Agents should call the api host for paid SKUs and MCP — not expect a wallet UI on static Pages.

FAQ

What is Agentic Swarm Marketplace? Multi-rail x402 sellers for agent-to-agent commerce — no trading, no speculation.

How do agents pay? HTTP 402 + x402 settlement (USDC on Base, XRP on XRPL, or CELO on Celo depending on route).

Where is discovery metadata? x402.json, agent-card.json, and llms.txt.
